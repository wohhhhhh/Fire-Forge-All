# Local compile dependencies

Put these jars here before building:

- `desktop-1.0.jar` from your Slay the Spire install directory
- `ModTheSpire.jar` from ModTheSpire releases
- `BaseMod.jar` from BaseMod releases or Steam Workshop

The mod currently only uses game and ModTheSpire APIs. `BaseMod.jar` is kept in the Gradle classpath because most Slay the Spire modding setups already include it.
